# Zephyr Sky (제퍼 스카이)

Sophisticated Minimalist Weather App with beautiful gradients.

## 주요 기능
- **현재 위치 날씨 조회:** 실시간 위치 기반의 정확한 기상 정보 제공.
- **전 세계 도시 검색:** 궁금한 전 세계 모든 도시의 날씨 확인.
- **마지막 위치 저장:** 앱 재실행 시 마지막으로 본 위치 자동 로드.
- **상태바 알림:** 앱을 열지 않아도 실시간 날씨를 상태바에서 확인.
- **현대적인 디자인:** 날씨 상태에 따라 변하는 유려한 그라데이션 UI.
