# AGENTS.md

AI 코딩 에이전트를 위한 범용 작업 규칙 및 프로젝트 관리 템플릿입니다.

## 🚀 개요

이 저장소는 AI 코딩 에이전트(Cursor, Windsurf, Gemini CLI 등)가 프로젝트 내에서 자율적으로, 그러나 안전하고 일관되게 작업할 수 있도록 돕는 표준 가이드라인인 `AGENTS.md`를 포함하고 있습니다.

## 🛠 주요 특징

- **Automation First**: 에이전트가 분석부터 커밋, 푸시까지 최대한 자동으로 수행하도록 유도합니다.
- **Safety Rails**: 파괴적인 명령이나 보안에 민감한 작업은 반드시 사용자 승인을 거치도록 제한합니다.
- **Standardized Communication**: 한국어 기반의 명확한 커뮤니케이션과 문서화 규칙을 제공합니다.
- **Workflow Integrity**: 작업 전 상태 확인부터 구현, 검증, 사후 보고까지의 전 과정을 체계화합니다.

## 📖 사용 방법

1. 이 저장소의 `AGENTS.md` 파일을 복사하여 여러분의 프로젝트 루트 디렉토리에 넣으세요.
2. 각 프로젝트의 `GEMINI.md`, `CLAUDE.md`, `.cursorrules` 등에서 이 파일을 참조하도록 설정하세요.
   ```text
   @AGENTS.md
   ```
3. `AGENTS.md` 내의 **프로젝트 설정값** 섹션을 해당 프로젝트에 맞게 수정하세요.

## 📜 주요 규칙 요약

- **구현 원칙**: 기존 스타일 준수, UI/로직 분리, 리소스 관리 철저.
- **검증 원칙**: 로컬 검증 후 GitHub Actions(CI)를 통한 최종 확인 지향.
- **문서화**: 코드 변경 시 `HISTORY.md`, `CHANGELOG.md` 등을 실시간 갱신.
- **보안**: 시크릿 노출 금지, 사용자 데이터 보호 최우선.

이 가이드라인은 계속해서 발전하고 있습니다. 더 좋은 규칙이나 개선 제안이 있다면 Issue 또는 PR을 통해 자유롭게 참여해 주세요.
