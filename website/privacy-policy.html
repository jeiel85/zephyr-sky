개인정보 처리방침

Zephyr Sky (제퍼 스카이)

최종 수정일: 2026년 5월 4일

1. 소개

Zephyr Sky(이하 "본 앱")는 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 개인정보 처리방침은 본 앱이 수집하는 정보와 그 사용 목적, 보관 방법에 대해 설명합니다.

2. 수집하는 정보

2.1 위치 정보

  • GPS 좌표: 현재 위치의 위도, 경도
  • 사용 목적: 현재 위치의 정확한 날씨 정보 제공
  • 수집 시점: 사용자가 위치 권한을 허용한 경우에만 수집

2.2 기기 정보

  • 저장소: 앱 설정, 즐겨찾기 위치, 캐시된 날씨 데이터
  • 사용 목적: 사용자 설정 유지 및 오프라인 기능 제공
중요: 본 앱은 사용자의 개인을 식별할 수 있는 정보(이름, 이메일, 전화번호 등)를 수집하지 않습니다.

3. 정보의 사용

수집된 정보는 다음 목적으로만 사용됩니다:

  • 현재 위치 기반 날씨 정보 제공
  • 사용자 설정(단위, 테마, 언어 등) 저장
  • 즐겨찾는 위치 목록 관리
  • 오프라인 모드에서의 캐시 데이터 제공

4. 제3자 제공

본 앱은 다음 외부 서비스와 연동됩니다:

4.1 Open-Meteo API

  • 제공 정보: 위도, 경도 좌표
  • 사용 목적: 날씨 및 대기질 데이터 조회
  • 개인정보: 좌표만 전송되며, 개인 식별 정보는 전송되지 않음

4.2 Geocoding API

  • 제공 정보: 검색어, 좌표
  • 사용 목적: 도시 검색 및 역지오코딩
광고 및 분석 도구: 본 앱은 광고 SDK나 사용자 행동 분석 도구(Google Analytics, Firebase Analytics 등)를 사용하지 않습니다.

5. 데이터 보관

  • 로컬 저장: 모든 데이터는 기기 내부 저장소에만 저장됩니다.
  • 서버 저장: 별도의 서버에 데이터를 저장하지 않습니다.
  • 캐시 데이터: 날씨 정보는 30분간 캐시되어 오프라인 모드를 지원합니다.

6. 사용자 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다:

  • 위치 권한 철회: 기기 설정에서 위치 권한을 비활성화할 수 있습니다.
  • 데이터 삭제: 앱을 삭제하면 모든 로컬 데이터가 삭제됩니다.
  • 캐시 삭제: 설정 화면에서 캐시된 데이터를 삭제할 수 있습니다.

7. 보안

본 앱은 다음과 같은 보안 조치를 취하고 있습니다:

  • 모든 외부 통신은 HTTPS를 통해 암호화됩니다.
  • 위치 데이터는 날씨 조회에만 사용되며, 다른 목적으로 사용되지 않습니다.
  • 개인정보를 수집하지 않으므로, 개인정보 유출 위험이 없습니다.

8. 어린이 개인정보 보호

본 앱은 13세 미만 어린이를 대상으로 하지 않으며, 어린이의 개인정보를 의도적으로 수집하지 않습니다.

9. 개인정보 처리방침 변경

본 개인정보 처리방침이 변경될 경우, 앱 업데이트 시 공지사항을 통해 알려드리며, 변경된 내용은 이 페이지에서 확인하실 수 있습니다.
