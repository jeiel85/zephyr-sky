A fully open-source weather application that respects your privacy. 

Features include:
- Real-time weather data from Open-Meteo (No API key required)
- Minimalist & Aesthetic UI with dynamic gradients
- Hourly and Daily forecasts
- Persistent Status Bar Notification (Location, Weather, Min/Max Temp)
- Global Location Search
- No trackers, No ads, Purely Open Source

Built with Flutter for high performance and beautiful UI. Ready for F-Droid and Google Play Store.
